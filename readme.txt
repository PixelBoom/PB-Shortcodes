Enables shortcodes to be used in PixelBoom WordPress Themes.

== Description ==

Easily add buttons, alerts, tabs, toggle, highlights and more – all with a nice shortcode generator located in the WordPress visual editor. After installing and activating the plugin you’ll see the shortcode generator button in your WordPress editor.

== Installation ==

Simply install as a normal WordPress plugin and activate.
